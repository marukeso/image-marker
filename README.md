# image-marker（画像にマーク：赤丸連番・星）

写真に **赤丸＋連番** や **星マーク** を置いて、PNGで書き出すためのシンプルなツールです。
指示出し・チェック・レビューなどで「この箇所」を番号で伝えたいときに使えます。

ビルド不要・サーバー不要。`index.html` をブラウザで開くだけで動きます。
画像はブラウザ内だけで処理され、どこにもアップロードされません。

**公開URL（GitHub Pages）：https://marukeso.github.io/image-marker/**
ブラウザで開けばそのまま使えます（インストール不要）。

## 特長

- **赤丸＋連番** … クリックした順に自動で番号が振られます
- **星マーク** … 番号なしの★を置けます
- **3枚同時** … 画像を横に3枚並べて扱え、番号は **3枚を通した連番** になります
- **開始番号の指定** … 1以外（例：5から）で始められます
- **取り消し / 全消し** … 直前の1つを取り消し（`Ctrl+Z`）、またはまとめて消去
- **PNGで保存** … 画像のあるパネルを1枚ずつ書き出し（`marked-1.png` …）
- マークのサイズは画像の大きさに合わせて自動調整

## 使い方

1. `index.html` をブラウザ（Chrome / Edge など）で開く
2. 各枠を **クリック** するか、画像を **ドラッグ＆ドロップ** して読み込む
3. ツールバーで「★ 星マーク」か「① 赤丸＋連番」を選ぶ
4. 画像上を **クリック** してマークを置く
5. 「PNGで保存」で書き出す

## 操作一覧

| 操作 | 内容 |
| --- | --- |
| 画像の読み込み | 空枠をクリック、または画像をドロップ |
| マークを置く | 画像をクリック（選択中のツールで設置） |
| 取り消し | 「取り消し」ボタン または `Ctrl+Z`（直前の1つ） |
| 全消し | すべてのマークを消去 |
| 開始番号 | 連番の開始値を指定 |
| PNGで保存 | 画像のあるパネルをそれぞれPNGで書き出し |

## 技術メモ

- HTML / CSS / JavaScript のみ（依存ライブラリなし・全部 `index.html` 1ファイル）
- Canvas API で画像とマークを描画
- 表示縮小ぶんを補正してクリック座標を実サイズに変換するため、大きい画像でも位置がずれません
